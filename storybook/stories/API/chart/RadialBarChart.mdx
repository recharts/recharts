The RadialBarChart can be used with the following child components: `<PolarAngleAxis />`, `<PolarRadiusAxis />`, `<PolarGrid />`, `<Legend />`, `<Tooltip />`, `<RadialBar />`, `<Customized />` or valid svg elements.
