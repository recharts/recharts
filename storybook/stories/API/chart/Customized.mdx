# Customized

## Parent Component

The Customized can be used within: `<LineChart />`, `<BarChart />`, `<ComposedChart />`, `<AreaChart />`, `<PieChart />`, `<RadarChart />`, `<RadialBarChart />`, `<ScatterChart />` or `<FunnelChart />`.

## Properties

Properties in the groups Other and Internal are not recommended to be used.
