## Description

<p>Cell can be wrapped by Pie, Bar, or RadialBar to specify attributes of each child. In Pie , for example, we can specify the attributes of each child node through data, but the props of Cell have higher priority</p>


## Parent Component

The Cell can be used within the following parent components:
- `<Bar/>`
- `<Funnel/>`
- `<Pie/>`
- `<RadialBar/>`
- `<Scatter/>`
