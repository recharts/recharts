## Description

<p>Cell component used to define colors and styles of chart elements.</p>
<p>This component is now deprecated and will be removed in Recharts 4.0.</p>
<p>Please use the <code>shape</code> prop or <code>content</code> prop on the respective chart components
to customize the rendering of chart elements instead of using <code>Cell</code>.</p>


## Parent Component

The Cell can be used within the following parent components:
- `<Bar/>`
- `<Funnel/>`
- `<Pie/>`
- `<RadialBar/>`
- `<Scatter/>`
