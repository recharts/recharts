## Description

<p>Renders a rectangle element. Unlike the rect SVG element, this component supports rounded corners
and animation.</p>
<p>This component accepts X and Y coordinates in pixels.
If you need to position the rectangle based on your chart&#39;s data,
consider using the ReferenceArea component instead.</p>
