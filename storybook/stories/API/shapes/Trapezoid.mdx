# Trapezoid

The Trapezoid is a shape that is valid in the context of any SVG as well as within the `<Customized />` component.

```jsx
<svg height={200} width="%100">
  <Trapezoid {...trapezoidProps} />
</svg>
```

or

```jsx
<Customized component={<Trapezoid {...trapezoidProps} />} />
```
