## Description

<p>Draws a line on the chart connecting two points.</p>
<p>This component, unlike line, is aware of the cartesian coordinate system,
so you specify the dimensions by using data coordinates instead of pixels.</p>
<p>ReferenceLine will calculate the pixels based on the provided data coordinates.</p>
<p>If you prefer to render using pixels rather than data coordinates,
consider using the line SVG element instead.</p>


## Parent Component

The ReferenceLine can be used within the following parent components:
- `<AreaChart/>`
- `<BarChart/>`
- `<ComposedChart/>`
- `<FunnelChart/>`
- `<LineChart/>`
- `<ScatterChart/>`

## Child Components

The ReferenceLine can be used with the following child components:
- `<Label/>`
