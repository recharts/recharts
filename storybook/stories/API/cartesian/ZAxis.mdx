# ZAxis

<Meta of={ComponentStories} />

<Canvas of={ComponentStories.API} layout="padded" />

## Description

<p>
  Virtual axis, does not render anything itself. Has no ticks, grid lines, or labels. Useful for dynamically setting
  Scatter point size, based on data.
</p>

## Parent Component

The ZAxis can be used within the following parent components:

- `<AreaChart/>`
- `<BarChart/>`
- `<ComposedChart/>`
- `<FunnelChart/>`
- `<LineChart/>`
- `<ScatterChart/>`
